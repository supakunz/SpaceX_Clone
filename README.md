## 🛰️ Installation

1. Clone this repository :

```bash
git clone https://github.com/SupakunZ/SpaceX_Clone.git
```

2. Navigate to the project folder and install dependencies :

```
cd SpaceX_Clone
npm install
```

3. Launch the application in development mode :

```
npm run dev
```
